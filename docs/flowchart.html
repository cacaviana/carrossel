Fase 1 — Entrada
USER Escolher Formato
Sidebar: Carrossel (1080x1350) · Post Unico (1080x1080) · Thumbnail YouTube (1280x720) · Funil (3 formatos combinados)
/ · /?formato=carrossel · /?formato=post_unico · /?formato=funil
↓
USER Selecionar Marca + Avatar
Marca carregada de GET /api/brands. Avatar: "Na capa" / "Livre" / "Sem avatar" (opcoes variam por formato)
↓
DECISAO Fonte do Conteudo
3 modos de entrada — cada um monta o "tema" de forma diferente
Texto Pronto
Preenche 1-7 slides manualmente (titulo + texto alternativo). Tema = texto do slide 1.
Ideia Livre
Textarea aberto (min 20 chars). Tema = texto digitado.
Por Disciplina
Seleciona disciplina + tecnologia + tema opcional. Tema = "D1 - XGBoost - meu tema".
↓
USER Criar Pipeline
Clica "Criar [Formato]". Envia tema + formato + marca + avatar + modo_entrada + slides (se texto pronto).
POST /api/pipelines/
Fase 2 — Pipeline de 7 Agentes
USER Painel do Pipeline
Wizard vertical com 7 etapas. Polling automatico a cada 3s. Mostra progresso em tempo real.
/pipeline/{id}
↓
AGENTE 1 Strategist
Analisa tema e cria briefing estruturado: angulo editorial, estrutura de slides, tom de voz, publico-alvo.
~15s · Claude API
↓
AP-1 Revisar Briefing
Usuario ve o briefing completo, pode editar o texto, ve tendencias detectadas.
/pipeline/{id}/briefing
Aprovar
Envia briefing (editado ou original). Dispara copywriter automaticamente. Volta pro painel.
Rejeitar e Regerar
Escreve feedback opcional. Strategist re-executa com o feedback. Pagina recarrega com novo briefing.
↺ Loop: rejeicao volta pro Strategist com feedback
↓
AGENTE 2 Copywriter
Gera copy completa: headline, narrativa, CTA, conteudo de cada slide (titulo + corpo + tipo).
~20s · Claude API
↓
AUTO Auto-aprovado pelo frontend
Polling detecta "aguardando_aprovacao" e aprova automaticamente. Dispara Hook Specialist.
↓
AGENTE 3 Hook Specialist
Gera 3 ganchos (A/B/C) com abordagens diferentes: resultado impossivel, contra-intuitivo, espelho, autoridade, urgencia.
~10s · Claude API
↓
AP-2 Revisar Copy + Hook
Seleciona 1 dos 3 hooks (obrigatorio). Pode editar headline, narrativa, CTA e conteudo de cada slide.
/pipeline/{id}/copy
Aprovar
Envia copy editada + hook selecionado. Dispara Art Director. Volta pro painel.
Regerar
Escreve feedback. Re-executa Hook Specialist (e possivelmente Copywriter). Recarrega com novos hooks.
↺ Loop: rejeicao volta pro Hook Specialist (ou Copywriter) com feedback
↓
AGENTE 4 Art Director
Cria prompt de imagem detalhado para cada slide. Respeita brand palette, cores e estilo visual da marca.
~15s · Claude API
↓
AP-3 Revisar Prompts Visuais
Ve/edita prompt de imagem de cada slide. Sidebar mostra paleta da marca e historico de preferencias visuais.
/pipeline/{id}/visual
Aprovar e Gerar Imagens
Envia prompts editados. Dispara Image Generator (~2min). Volta pro painel com barra de progresso.
Rejeitar
Art Director re-executa. Recarrega prompts. Fica na mesma pagina.
↺ Loop: rejeicao re-gera prompts visuais
↓
AGENTE 5 Image Generator
Gera imagens via Gemini API. 3 slides simultaneos (semaforo). Pro para capa/CTA, Flash para conteudo. Retry com fallback.
~1-2min · Gemini API · Barra de progresso "Slide N/Total"
↓
AUTO Auto-aprovado pelo frontend
Polling detecta imagens prontas e aprova automaticamente. Dispara Brand Gate.
↓
SKILL Brand Gate
Deterministico (sem LLM). Valida cores e elementos da marca. Aplica foto redonda + logo via Pillow.
~5s · Pillow
↓
AGENTE 7 Content Critic
Avalia tudo: clarity, impact, originality, scroll_stop, cta_strength. Score final + decisao (approved/needs_revision).
~10s · Claude API
↓
AUTO Pipeline Completo
Todos os 7 passos verdes. Botao "Ver Imagens" aparece. Score final exibido.
Fase 3 — Editor Visual
USER Editor de Slides
Preview dos slides gerados. Clique pra posicionar logo. Controles de tamanho (rodape/central), borda, cor.
/editor?pipeline={id}&brand={slug}
↓
POR SLIDE Slide OK?
Usuario navega slide a slide, avaliando imagem e texto.
OK — Proximo slide
Slide esta bom. Navega pro proximo. No ultimo slide: "Baixar PDF".
Precisa ajuste
Texto errado? "Corrigir texto" — Gemini reescreve texto na imagem (ate 3 tentativas com OCR).
Imagem ruim? "Regenerar slide" — gera imagem nova do zero via Gemini.
↺ Loop por slide ate todos ficarem OK
↓
USER Posicionar Logo em Todos
"Igual em todos" — aplica posicao do logo atual em todos os slides. Upload de logo novo se necessario.
Fase 4 — Export
USER Baixar PDF
No ultimo slide, clica "Baixar PDF". Backend compoe slides + logo nas posicoes certas e gera PDF.
POST /api/editor/salvar-pdf
↓
Carrossel Pronto
PDF baixado. Slides com logo, texto correto, identidade visual da marca. Pronto pra postar no LinkedIn.
Fluxo de Erro (qualquer agente)
ERRO Agente Falhou
Timeout, API key invalida, rate limit, resposta inesperada. Step fica vermelho no painel.
Retomar
Re-executa o mesmo agente do zero. Pipeline volta pra "pendente".
Abrir no Editor
Pula agentes restantes e vai direto pro editor com as imagens que ja tem.
Cancelamento
CANCELAR Pipeline Cancelado
Botao "Cancelar" no painel. Confirmacao em modal. Pipeline salvo no historico como "cancelado". Nao pode ser retomado.
POST /api/pipelines/{id}/cancelar → goto /historico
